# CLAUDE.md

승인 엔진 애플리케이션 계층의 요청·응답 DTO(record).

## 파일
- `ApprovalInboxItemResponse.java` — 대기함 목록 항목(API-COM-003)
- `ApprovalDetailResponse.java` — 인스턴스 상세(API-COM-004, 차수별 역할 결정 현황 포함)
- `ApprovalDecisionRequest.java` — 승인/반려 결정 요청(decision, reason)
- `ApprovalDecisionResultResponse.java` — 결정 처리 결과(차수·인스턴스 상태)
