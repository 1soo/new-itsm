# CLAUDE.md

esm 도메인의 REST 컨트롤러.

## 파일
- `EsmCatalogController.java` — `/api/v1/esm/catalog-items` 부서 카탈로그 목록/상세/생성/수정 API
- `EsmRequestController.java` — `/api/v1/esm/requests` 부서 요청 제출/목록/상세/상태전이/코멘트 API
- `EsmHrCaseController.java` — `/api/v1/esm/hr-cases` HR 케이스 접수/목록/상세/상태전이 API
- `EsmChecklistController.java` — `/api/v1/esm/checklists`, `/api/v1/esm/checklist-tasks` 체크리스트 상세·내 하위 작업 목록·완료 처리 API
- `EsmMetricsController.java` — `/api/v1/esm/metrics` ESM 지표 API
