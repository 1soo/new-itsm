# CLAUDE.md

asset 도메인의 리포지토리 인터페이스.

## 파일
- `AssetRepository.java` — 자산(Asset) 저장·조회·검색(만료 임박 포함)
- `AssetAttributeRepository.java` — 유형별 속성(AssetAttribute) 저장·조회·삭제
- `AssetLifecycleHistoryRepository.java` — 생애주기 이력(AssetLifecycleHistory) 저장·조회
- `ConfigurationItemRepository.java` — 구성 항목(ConfigurationItem) 저장·조회·검색
- `CiRelationRepository.java` — CI 관계(CiRelation) 저장·조회
