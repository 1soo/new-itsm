# CLAUDE.md

재사용 컴포넌트 계층. 저수준 프리미티브(ui) → 도메인 무관 공통 패턴(common) → 앱 레이아웃 셸(layout) 순으로 조합. 모두 프레젠테이션 위주이며, 데이터·권한·라우팅은 기능 레이어(FE)가 주입.

## 하위 디렉토리
- `ui/` — shadcn/ui 기반 공통 UI 프리미티브(Radix + 디자인 토큰).
- `common/` — 프로젝트 공통 컴포넌트(상태 표시·목록/상세 셸·차트·동적 폼 등). ui를 조합.
- `layout/` — 애플리케이션 레이아웃 셸(헤더·사이드바·푸터).
