### TC-PRB-002 · 상태 전이 버튼 라벨(동작 동사형) 표시
- 근거: @docs/02_plan/screen/problem.md SCR-PRB-003 "전이 버튼 라벨" 표
- 전제: pm@itsm.local로 신규 문제 등록
- 절차:
  1. CLASSIFICATION 버튼 텍스트 확인("분류 시작") 후 클릭
  2. INVESTIGATION 버튼 텍스트 확인("조사 시작") 후 클릭
  3. KNOWN_ERROR 버튼 텍스트 확인("알려진 오류로 등록") 후 클릭
  4. WORKAROUND 버튼 텍스트 확인("워크어라운드 등록") 후 클릭(워크어라운드 내용 입력)
  5. RESOLVED_CLOSED 버튼 텍스트 확인("종료 처리") 후 클릭
- 기대 결과: 각 단계 버튼 텍스트가 도착 상태명이 아닌 표의 동작 동사형 라벨과 정확히 일치
