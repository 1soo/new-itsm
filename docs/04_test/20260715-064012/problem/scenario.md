## 시나리오

### TC-PRB-101 · 문제 등록 및 우선순위 자동 산정
- 근거: @docs/01_analyze/feature/problem.md (SCR-PRB-001/002, 영향도×긴급도 → 우선순위 계산)
- 전제: `/problems` 목록이 빈 상태(트랜잭션 데이터 미시딩, 설계상 정상)
- 절차:
  1. "문제 등록" → 요약/설명/출처(반응)/영향도(보통)/긴급도(보통) 입력
  2. 입력 완료 시 "우선순위 미리보기" 자동 계산값 확인
  3. 등록 → 상세 화면 이동 확인
- 기대 결과: 우선순위 P3로 자동 산정되어 미리보기에 표시, 등록 성공, 콘솔 에러 없음
