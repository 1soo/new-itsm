# 통합 테스트 시나리오 — knowledge (런타임 업그레이드 후 기능 회귀)

## 사전 조건
- auth 도메인 로그인 성공 전제. 신규 기능 검증 아님(기존 동작 회귀 확인).

## 시나리오

### TC-KM-101 · 기사 작성 및 카테고리 시드 데이터 확인
- 근거: @docs/01_analyze/feature/knowledge.md (SCR-KM-001/002, 카테고리 3종)
- 전제: `/knowledge` 목록이 빈 상태(트랜잭션 데이터 미시딩, 설계상 정상)
- 절차:
  1. "기사 작성" → 제목/본문 입력
  2. 카테고리 콤보박스에서 시드된 3개 카테고리(네트워크/계정·권한/결제) 노출 확인 후 "네트워크" 선택
  3. 저장
- 기대 결과: 기사 저장 성공, 편집 화면(`/knowledge/1/edit`)으로 이동, 콘솔 에러 없음
