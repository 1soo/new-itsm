# 화면 설계서 — 404 오류

> 도메인: common(error) · 버전: 0.1 · 작성일: 2026-07-09

## 1. 개요

존재하지 않는 경로 접근 시 표시하는 404 화면을 정의한다. 공통 팔레트를 따른다.

## 2. 색상 팔레트

[common.md](common.md)의 공통 팔레트를 그대로 사용한다.

| 구분 | 색상 (HEX) | 용도 |
|------|-----------|------|
| Base | #2563EB | "홈으로" 버튼 |
| Neutral Dark | #1F2937 | 안내 텍스트 |
| Muted | #9CA3AF | 보조 설명 |

## 3. 화면 목록

| 화면 ID | 화면명 | 관련 요구사항/기능 | 설명 |
|---------|--------|--------------------|------|
| SCR-ERR-404 | 404 Not Found | 전 도메인 | 잘못된 경로 안내 |

## 4. 화면 상세

### SCR-ERR-404 · 404 Not Found

- **목적**: 없는 경로/리소스 접근 시 안내하고 정상 경로로 유도.
- **레이아웃**: 중앙 정렬. 큰 "404" 표기 / 안내 문구 / "홈(대시보드)으로" 버튼.
- **구성 요소**:
  | 요소 | 유형 | 설명 | 색상 |
  |------|------|------|------|
  | 404 표기 | 텍스트 | 대형 숫자 | Muted |
  | 안내 문구 | 텍스트 | "요청하신 페이지를 찾을 수 없습니다" | Neutral Dark |
  | 홈으로 버튼 | 버튼 | 대시보드/포털로 이동 | Base |
- **상태 · 인터랙션**: 클라이언트 라우팅 미매칭 시 표시. 버튼 클릭 시 역할 기본 홈으로 이동.
- **연관 API**: 없음
