# 요구사항 정의서 — 문제 관리 (Problem Management)

> 도메인: problem · 버전: 0.1 · 작성일: 2026-07-09

## 1. 개요

인시던트의 근본 원인(root cause)을 식별·관리하여 재발을 예방한다. 문제 등록, 6단계 프로세스(탐지→분류/우선순위→조사/진단→알려진 오류 기록→워크어라운드→해결/종료), 알려진 오류 데이터베이스(KEDB), 인시던트·변경·지식 연계를 제공한다. (출처: `problem-management.md`)

## 2. 범위

### 포함 (In Scope)

- 문제 등록 및 티켓 필드(신고자·담당자·우선순위·영향도·긴급도·출처·조사사유·근본원인·워크어라운드·구성요소·연결이슈 등)
- 문제 프로세스 6단계 상태 관리
- 우선순위 산정(영향도 × 긴급도)
- 근본 원인 분석(RCA) 기록 및 5 Whys 지원
- 알려진 오류(Known Error) 레코드 생성·KEDB 조회
- 워크어라운드 등록(지식베이스 문서화 연계)
- 인시던트 연결, 변경 요청 연계, 후속 조치 추적
- 반응적/선제적 문제 구분

### 미포함 (Out of Scope)

- 문제 관리 전용 KPI 대시보드(요청·인시던트 도메인 지표와 별개 대시보드는 미포함)
- 자동 문제 탐지(모니터링 연동)

## 4. 인수 기준 (Acceptance Criteria · EARS)

### REQ-PRB-001

- (Event-driven) **WHEN** 사용자가 요약·설명·출처·조사사유 등을 포함해 문제를 등록하면, 시스템은 문제를 생성하고 식별키를 반환해야 한다.
- (Unwanted) **IF** 필수 필드(요약)가 누락되면, **THEN** 시스템은 생성을 거부해야 한다.

### REQ-PRB-002

- (Event-driven) **WHEN** 담당자가 문제 단계를 전이(탐지→분류→조사/진단→…→종료)하면, 시스템은 허용된 전이만 반영해야 한다.
- (Unwanted) **IF** 순서에 맞지 않는 전이를 시도하면, **THEN** 시스템은 이를 거부해야 한다.

### REQ-PRB-003

- (Event-driven) **WHEN** 영향도와 긴급도가 입력되면, 시스템은 우선순위를 산정해 표시해야 한다.

### REQ-PRB-004

- (Event-driven) **WHEN** 사용자가 근본 원인(및 5 Whys 사슬)을 기록하면, 시스템은 이를 문제에 저장해야 한다.
- (Ubiquitous) 시스템은 근본 원인 기록 시 사람(개인)을 근본 원인으로 지정하도록 강제하지 않아야 한다(blameless 지향).

### REQ-PRB-005

- (Event-driven) **WHEN** 사용자가 근본원인과 워크어라운드를 포함한 알려진 오류 레코드를 생성하면, 시스템은 이를 KEDB에 등록해야 한다.
- (Event-driven) **WHEN** 사용자가 KEDB를 키워드로 조회하면, 시스템은 매칭되는 알려진 오류를 반환해야 한다.

### REQ-PRB-006

- (Event-driven) **WHEN** 사용자가 워크어라운드를 등록하면, 시스템은 이를 문제에 저장하고 지식베이스 기사와 연결할 수 있어야 한다.

### REQ-PRB-007

- (Event-driven) **WHEN** 사용자가 인시던트를 문제에 연결하면, 시스템은 두 항목을 링크하고 양방향으로 표시해야 한다.
- (Unwanted) **IF** 존재하지 않는 인시던트를 연결하면, **THEN** 시스템은 400 오류를 반환해야 한다.

### REQ-PRB-008

- (Event-driven) **WHEN** 사용자가 문제에서 변경 요청을 생성/연계하면, 시스템은 문제와 변경을 링크해야 한다.

### REQ-PRB-009

- (Ubiquitous) 시스템은 문제에 연결된 후속 조치 항목의 상태(진행중/완료)를 추적할 수 있어야 한다.

### REQ-PRB-010

- (Event-driven) **WHEN** 사용자가 문제를 종료하면, 시스템은 상태를 종료로 전환하고 이력에 기록해야 한다.
- (Unwanted) **IF** 미해결 후속 조치가 남은 상태로 종료하려 하면, **THEN** 시스템은 경고를 표시해야 한다.
